- Auth regression testing: automated integration test covers sign-up → sign-in → session persists on refresh → sign-out → sign-in; manually verify on both new and pre-existing accounts; test forgot-password flow on any password storage change. Confidence: 0.85
- Auth regressions are Critical severity, never deferred to next session. Confidence: 0.85
- Super admin: mandatory 2FA/MFA (TOTP minimum), separate shorter session lifetime (15-30 min), IP/device visibility logged, no shared credentials, impersonation is logged, treat workspace and admin login as separate code paths. Confidence: 0.85

- Embed code: generate real working snippet scoped to asset's ID/workspace, support both iframe and script loader embed, embed by reference/ID so updates reflect without re-embedding. Confidence: 0.70
- Publish workflow: real Draft → Review → Live states backed by actual published/unpublished content (Draft not publicly reachable, Live reachable within seconds), version history with last 10 saves and one-click restore with real diffing, every publish triggers SEO indexing pipeline. Confidence: 0.70
- Domain/embed/publish features tested exactly as user would: real DNS zone, real HTML file with embedded snippet, incognito browser to verify Live URL. Confidence: 0.70

Confidence: 0.85
