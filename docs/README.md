*   [**PROJECT_PHASE_STATUS.md**](file:///home/suite.digitpenhub.com/digitpenhub-suite/docs/current/PROJECT_PHASE_STATUS.md) - Phase status ledger representing completed milestones (Phase 0 through Phase 4 testing).
*   [**MODULE_VERIFICATION_STATUS.md**](file:///home/suite.digitpenhub.com/digitpenhub-suite/docs/current/MODULE_VERIFICATION_STATUS.md) - Status and verification checklist of all Priority-1 core business modules.
*   [**DEPLOYMENT_GUIDE.md**](file:///home/suite.digitpenhub.com/digitpenhub-suite/docs/current/DEPLOYMENT_GUIDE.md) - Deployment instructions, local setup scripts guide, and staging requirements.
